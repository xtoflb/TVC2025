# Rappel des commandes Linux
## Commandes de base
- Lister le contenu d'un dossier
```bash
ls -l
```
- Changer de dossier
```bash
cd /home/stagiaire
```
- Indiquer le dossier courant
```bash
pwd
```
- Créer un dossier
```bash
mkdir ~/dossier_Test
```
- Supprimer un fichier ou un dossier
```bash
rm -R ~/dossier_Test
```
- Copier un fichier ou un dossier
```bash
cp -R ~/Téléchargements ~/Téléchargements2
```
- Déplacer un fichier ou un dossier
```bash
mv -R ~/Téléchargements2 ~/Documents/
```
- Trouver un fichier
```bash
find -name toto.txt
```
